# AI 댓글 부자연스러움 원인 분석

## 🔍 발견된 문제점

### 1. **좋은_댓글_예시가 너무 적음** ⚠️ 가장 큰 문제
- 현재: **단 3개만** 있음
- 문제: Few-shot learning을 위해서는 **최소 10-20개**는 필요함
- 결과: AI가 학습할 패턴이 부족해서 반복적이고 단순한 댓글 생성

### 2. **예시가 너무 단순하고 반복적**
- 현재 예시: "대박이네요", "아쉽네요", "힘내요"
- 문제: 
  - 모두 비슷한 패턴 (단어 + "요" 어미)
  - 본문 내용을 제대로 반영하지 못함
  - 실제 커뮤니티에서는 더 다양하고 구체적인 댓글이 나옴
- 예시:
  - ❌ "정배 찍었는데 다 터졌어요" → "대박이네요" (너무 단순)
  - ✅ "정배 찍었는데 다 터졌어요" → "부럽네요" (더 구체적)

### 3. **본문과 댓글의 연결이 약함**
- 문제: 댓글이 본문의 구체적인 내용을 반영하지 못함
- 예시:
  - 본문: "오늘 정배 찍었는데 다 터졌어요"
  - 현재 댓글: "대박이네요" (너무 일반적)
  - 개선: "부럽네요" 또는 "축하해요" (본문의 "터졌어요"를 반영)

### 4. **상황별 댓글 예시가 제한적**
- 현재: 4가지 상황만 (절망_후회, 기쁨_성공, 아쉬움, 슬픔_힘듦)
- 문제: 실제로는 더 다양한 상황이 있음
- 부족한 상황:
  - 중립적/무관심한 글
  - 질문하는 글
  - 정보 공유하는 글
  - 불만/비판하는 글

### 5. **도박 용어는 많지만 실제 사용 예시 부족**
- 문제: 용어 정의는 많지만, 실제 댓글에서 어떻게 자연스럽게 사용하는지 예시가 부족
- 예시:
  - ❌ "정배 찍었는데 터졌어" → "정배 찍었는데 아쉽네요" (용어만 반복)
  - ✅ "정배 찍었는데 터졌어" → "부럽네요" (용어를 이해하고 자연스럽게 반응)

### 6. **기존 댓글 흐름 반영 부족**
- 문제: 기존 댓글들의 스타일을 제대로 분석하지 못함
- 예시:
  - 기존 댓글: ["축하해요", "부럽네요"] → 모두 존댓말
  - 생성 댓글: "대박이네요" (맞음)
  - 하지만 기존 댓글이 더 구체적이면 댓글도 더 구체적이어야 함

## 💡 개선 방안

### 1. 좋은_댓글_예시를 20개 이상으로 확대
- 다양한 상황 추가
- 다양한 말투 추가
- 다양한 스타일 추가

### 2. 더 구체적이고 자연스러운 예시 추가
- 본문 내용을 제대로 반영하는 댓글
- 기존 댓글 흐름을 자연스럽게 이어가는 댓글

### 3. 상황별 댓글 예시 확대
- 더 많은 상황 추가
- 각 상황별로 더 다양한 댓글 추가

### 4. 도박 용어를 자연스럽게 사용하는 예시 추가
- 용어를 단순히 반복하는 것이 아니라
- 용어의 맥락을 이해하고 자연스럽게 반응하는 예시

## 🎯 즉시 개선 가능한 사항

1. **좋은_댓글_예시를 10-20개로 확대**
2. **더 구체적이고 자연스러운 예시 추가**
3. **다양한 상황의 예시 추가**
4. **본문 내용을 제대로 반영하는 예시 추가**
