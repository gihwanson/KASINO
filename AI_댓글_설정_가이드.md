# AI 댓글 생성 설정 가이드

## 🤖 AI 댓글 기능

이제 매크로가 게시글 본문을 읽고, AI를 사용해서 관련된 자연스러운 댓글을 자동으로 생성합니다!

## ✨ 특징

- ✅ 게시글 본문을 자동으로 읽어서 분석
- ✅ 본문과 관련된 자연스러운 댓글 생성
- ✅ 10글자 이내로 작성
- ✅ ~요 체나 반말체로 친근하게 작성
- ✅ 격식 없는 자연스러운 톤

## 🔑 OpenAI API 키 설정

### 1. API 키 발급

1. https://platform.openai.com 접속
2. 회원가입 또는 로그인
3. API Keys 메뉴로 이동
4. "Create new secret key" 클릭
5. API 키 복사

### 2. .env 파일에 추가

`.env` 파일을 열어서 다음 줄을 추가:

```env
OPENAI_API_KEY=여기에_API_키_입력
```

**예시:**
```env
OPENAI_API_KEY=sk-1234567890abcdefghijklmnopqrstuvwxyz
```

### 3. 저장 후 실행

API 키를 설정하면 자동으로 AI 댓글이 생성됩니다!

## 💰 비용 안내

- OpenAI API는 사용량에 따라 과금됩니다
- GPT-3.5-turbo 모델 사용 (비용 효율적)
- 댓글 하나당 약 $0.0001 ~ $0.0002 정도
- 1000개 댓글 작성 시 약 $0.1 ~ $0.2 정도

## ⚙️ API 키 없이 사용하기

API 키를 설정하지 않으면 기본 댓글이 사용됩니다:
- "좋은 글 감사합니다!"
- "유용한 정보네요."
- 등등...

## 🎯 댓글 예시

**게시글 본문:** "오늘 돈까스 먹었는데 진짜 맛있었어요!"

**생성된 댓글:**
- "나도 돈까스 좋아해요~"
- "맛있겠다요!"
- "돈까스 최고요"

## 🆘 문제 해결

### API 키 오류
- API 키가 올바른지 확인
- OpenAI 계정에 충분한 크레딧이 있는지 확인

### 댓글 생성 실패
- 인터넷 연결 확인
- API 키가 유효한지 확인
- 실패 시 자동으로 기본 댓글 사용

## 💡 팁

- API 키는 절대 공유하지 마세요!
- `.env` 파일을 Git에 커밋하지 마세요
- 비용을 절약하려면 `MAX_POSTS` 값을 줄이세요
