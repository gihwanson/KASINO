# 처음 사용하기 가이드

## 🚀 3단계로 시작하기

### 1단계: 초기 설정 파일 만들기

**방법 1: 자동 설정 (추천)**
- `설정하기.bat` 파일을 더블클릭
- 자동으로 `.env` 파일이 생성됩니다

**방법 2: 수동 설정**
1. `.env.example` 파일을 복사
2. 이름을 `.env`로 변경

### 2단계: 로그인 정보 입력

1. `.env` 파일을 메모장으로 열기
   - 우클릭 → 연결 프로그램 → 메모장
   
2. **메모장에서 한글이 깨져 보이면:**
   - 파일 → 다른 이름으로 저장
   - 인코딩을 **"UTF-8"**로 선택
   - 저장

3. 다음 두 줄만 수정:
   ```env
   LOGIN_USERNAME=INPUT_YOUR_ID_HERE    ← 여기에 아이디 입력
   PASSWORD=INPUT_YOUR_PASSWORD_HERE    ← 여기에 비밀번호 입력
   ```

4. 저장 (Ctrl + S)

**예시:**
```env
LOGIN_USERNAME=myusername
PASSWORD=mypassword123!
```

### 3단계: 프로그램 실행

```bash
python macro_bot.py
```

또는 `실행하기.bat` 파일을 더블클릭

## ✅ 완료!

이제 프로그램이 자동으로 로그인하고 댓글을 작성합니다.

## 📝 추가 설정 (선택사항)

원한다면 `.env` 파일에서 다음 항목도 수정할 수 있습니다:

```env
# 최대 처리할 게시글 수
MAX_POSTS=5

# 게시글 처리 순서
POST_ORDER=latest    # 최신순
# 또는
POST_ORDER=random    # 랜덤
```

## 🆘 문제 해결

### .env 파일을 찾을 수 없어요
- 파일 탐색기에서 "숨김 항목" 표시하기
- 또는 `설정하기.bat` 실행

### 파일을 저장할 수 없어요
- 파일이 다른 프로그램에서 열려있는지 확인
- 메모장을 관리자 권한으로 실행 후 다시 시도

### 한글이 깨져요
- 파일 저장 시 인코딩을 UTF-8로 선택
- 또는 `.env.example` 파일이 영어로 작성되어 있으므로 그대로 사용 가능

## 💡 팁

- 아이디와 비밀번호만 수정하면 됩니다
- 나머지 설정은 기본값 그대로 사용 가능합니다
- `.env.example` 파일은 영어로 작성되어 있어 인코딩 문제가 없습니다
