# 파일 정리 가이드

## 타 사용자에게 배포할 때 포함할 파일

### 필수 파일
- `macro_bot.py` - 메인 프로그램
- `requirements.txt` - 필요한 패키지 목록
- `설정하기.bat` - 초기 설정
- `실행하기.bat` - 프로그램 실행
- `env.example` - 설정 예시 파일
- `update_env.py` - 설정 업데이트 스크립트

### 선택 파일 (가이드)
- `README_간단버전.md` - 간단한 사용 가이드

---

## 불필요한 파일 (배포 시 제외)

### 개발/테스트용
- `API_키_테스트.bat`, `API_키_테스트.py` - 테스트용
- `pip_설치.bat` - pip 설치용 (개발자용)
- `실행파일_만들기.bat` - 실행 파일 생성용 (개발자용)

### 상세 가이드 (선택사항)
- `AI_댓글_설정_가이드.md`
- `API_키_발급_가이드.md`
- `API_할당량_문제_해결.md`
- `Gemini_API_키_발급_가이드.md`
- `처음_사용하기.md` (간단 버전이 있으므로 선택)

### 개발자용 문서
- `배포용_압축_체크리스트.txt`
- `사용자_배포_가이드.md`

### 기타
- `package-lock.json` - 불필요
- `Lib/` - Python 패키지 (설치 시 자동 생성)
- `Scripts/` - Python 스크립트 (설치 시 자동 생성)

---

## 정리 방법

1. **개발자용 파일은 별도 폴더로 이동** (예: `개발자용/`)
2. **또는 삭제** (필요시 다시 생성 가능)
3. **Lib/, Scripts/ 폴더는 .gitignore에 추가**
