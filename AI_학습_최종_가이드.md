# AI 학습 최종 가이드

## ✅ 완료된 통합

**모든 AI 학습 데이터가 `AI_프롬프트_설정.json` 파일 하나에 통합되었습니다!**

### 포함된 내용:
1. ✅ 기본 규칙 (게시판 특성, 댓글 스타일, 최대 길이 등)
2. ✅ 금지 단어 목록
3. ✅ 좋은 댓글 예시 (Few-shot Learning)
4. ✅ 나쁜 댓글 예시 (피해야 할 패턴)
5. ✅ 상황별 댓글 예시
6. ✅ **도박 용어 사전 (100개 이상 용어)** ← 새로 통합됨!
7. ✅ 프롬프트 개선 내용

---

## 📝 사용 방법

### 개발자 (당신)
1. `AI_프롬프트_설정.json` 파일만 열기
2. 원하는 섹션 수정:
   - `좋은_댓글_예시`: 좋은 댓글 예시 추가
   - `나쁜_댓글_예시`: 피해야 할 댓글 추가
   - `도박_용어_사전`: 도박 용어 추가/수정
   - `금지_단어`: 금지 단어 추가
3. 저장
4. 매크로봇 실행 → 자동으로 반영됨!

### 타사용자 배포
1. `실행파일_만들기.bat` 실행
2. 생성된 파일들:
   - `매크로봇.exe`
   - `AI_프롬프트_설정.json` ← **이 파일 하나에 모든 학습 데이터 포함!**
   - 기타 설정 파일들
3. 타사용자에게 전달
4. 타사용자가 `매크로봇.exe` 실행 → AI 학습 설정 자동 적용!

## ✅ 확인 방법

프로그램 실행 시:
```
[AI] 프롬프트 설정 로드 완료 (좋은 예시: 3개, 나쁜 예시: 3개)
[AI] 도박 용어 사전 로드 완료 (100개 이상 용어)
```

이 메시지가 보이면 `AI_프롬프트_설정.json` 파일이 정상적으로 로드된 것입니다!

## 💡 핵심 정리

1. **하나의 파일로 모든 학습 관리**: `AI_프롬프트_설정.json` 하나만 수정하면 됩니다
2. **자동 반영**: 파일 수정 후 저장하면 매크로봇 실행 시 자동으로 적용됩니다
3. **타사용자 배포 간편**: 실행파일과 함께 `AI_프롬프트_설정.json` 하나만 전달하면 됩니다
4. **더 이상 별도 파일 불필요**: `도박용어_사전.json`, `ai_learning_data.json` 등은 더 이상 필요 없습니다
