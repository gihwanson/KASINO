# AI 학습 간단 가이드

## ✅ 네, 맞습니다!

### 1. `AI_프롬프트_설정.json` 파일만 수정하면 자동으로 학습됩니다

- 파일을 수정하고 저장하면 됩니다
- 매크로봇 실행 시 자동으로 이 파일을 읽어서 사용합니다
- 별도의 학습 프로그램 실행 불필요

### 2. 실행파일 만들기 시 자동으로 포함됩니다

- `실행파일_만들기.bat` 실행 시 다음 파일들이 자동으로 포함됩니다:
  - `AI_프롬프트_설정.json` ✅
  - `도박용어_사전.json` ✅
  - `env.example` ✅
  - `설정하기.bat` ✅

---

## 📝 사용 방법

### 개발자 (당신)
1. `AI_프롬프트_설정.json` 파일 열기
2. `좋은_댓글_예시` 배열에 원하는 댓글 예시 추가
3. 저장
4. 매크로봇 실행 → 자동으로 반영됨!

### 타사용자
1. `실행파일_만들기.bat` 실행
2. 생성된 `매크로봇.exe`와 함께 다음 파일들이 포함됨:
   - `AI_프롬프트_설정.json` (AI 학습 설정)
   - `도박용어_사전.json` (도박 용어)
   - `env.example` (환경 변수 예시)
   - `설정하기.bat` (설정 파일)
3. 타사용자가 `설정하기.bat` 실행하여 로그인 정보 입력
4. `매크로봇.exe` 실행 → AI 학습 설정 자동 적용!

---

## 🔄 작동 원리

```
1. 매크로봇 실행
   ↓
2. AI_프롬프트_설정.json 파일 자동 로드
   ↓
3. 좋은 댓글 예시, 나쁜 댓글 예시, 금지 단어 등 모든 설정 적용
   ↓
4. AI가 이 설정을 기반으로 댓글 생성
```

---

## 💡 중요 사항

### 실행파일 만들기 전에
- `AI_프롬프트_설정.json` 파일을 최종 버전으로 수정하세요
- 실행파일 만들기 후에는 이 파일이 함께 배포됩니다

### 타사용자 배포 시
- `매크로봇.exe`와 함께 `AI_프롬프트_설정.json` 파일도 함께 전달됩니다
- 타사용자가 이 파일을 수정하면 AI 댓글 스타일이 변경됩니다
- (선택사항) 타사용자가 수정하지 못하도록 하려면 파일을 읽기 전용으로 설정하세요

---

## ✅ 확인 방법

프로그램 실행 시 다음 메시지가 보이면 정상입니다:

```
[AI] 프롬프트 설정 로드 완료 (좋은 예시: 3개, 나쁜 예시: 3개)
[AI] 도박 용어 사전 로드 완료 (100개 이상 용어)
```

이 메시지가 보이면 `AI_프롬프트_설정.json` 파일이 정상적으로 로드된 것입니다!
